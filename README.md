RL Task

===

Setup instructions:

1. Clone the repository:

   ```
   git clone https://github.com/ivanadamov-appstreams/RL_Task.git
   ```

2. Navigate to the project directory:

   ```
   cd RL_Task
   ```

3. Set up `ANTHROPIC_API_KEY` environment variable:

   ```
   export ANTHROPIC_API_KEY=your_api_key_here
   ```

4. Run the agent:
   ```
   uv run python -m tasks.results.demo
   ```

## Execution Modes

As I faced troubles with token usage, only sequentional executions are supported. You can change the number of runs and the timeout between executions in demo files.

## tasks

There are multiple demo tasks, each following the same structure

- prompt.md
- task.py
- tools
- grader
- demo.py

## Main task - results

In data directory there is a file with matches results. They are with cyrilic letters as this is real exported data from local football league.

### The task:

Important notes:

- there are nested and OR conditions(the condition can be more complex in order to lower success rate)
- The model should not use Python expression(otherwise the task for the model is to write Python code which will be done easily)
- csv file is loaded initally, otherwise it will run out of tokens

Additional options for making the task more complex

- Use python_expression but add the number of letters of team name in the condition. It will be hard for the model to write such Python code
- Add noise in data - symbols(: instead - in match result), symbols in team names, additional columns

#### The prompt

```md
Find all games that satisfy at least one condition:

1. Away team won.
2. Home team won and away team scored at least one goal.
3. Home team won and total goals < 3.

Dataset:

{dataset}

Return list of dicts: date, home, result, away, round, season for the matching games.

Work through the dataset directly in your response (no tools), show reasoning, and call submit_answer with the final list.
```

Reasons

Factological:

- In one of the the results, there is a match which is not expected to be into
- Missing games

Formatting

- Missing letters in team names
- Changes in home/away fields in structure
